Description:
The "task_manager" project is a simple command-line task management application. It allows users to register, add tasks, view all tasks, view tasks assigned to them, display statistics, and generate reports. Users can register with a username and password, add tasks with details such as title, description, and due date, mark tasks as complete, edit tasks, and generate overview reports.

Importance:
Task management is crucial for individuals and teams to organize their work effectively, prioritize tasks, and ensure timely completion. This project provides a basic tool for managing tasks efficiently, helping users stay organized and focused on their objectives.
